    Title

    LetsJam.com

    Goal

    To inspire people get better at playing bass, and how to get better to play with a group of musicians.

    Target Audience

    The site is intended to reach people who are interested in knowing more about music, moreso bass, and to show bassists that you can do more than just pluck some notes and some strings.

    Main Features

    Web site with links to information, stories and pictures of bassplayers that inspire me to be a better bassplayer everyday. A gallery of images will provide a sense of the bass I have and bass I would love to have one day.

    Personality

    Bold, majestic, appealing, simple, clear, urgent.

    The site will help musicians, specially bassplayers, feel more passionate about learning, mastering this majestic instrument that I'm very passionate about the BASS.

    Limiting Factors

    Website will have an image gallery with large pictures, maybe some videos so internet capabilities should be fairly current. Audience should have some experience with Internet but does not need to be at the intermediate or expert level.

    Styles guide

    Background

    Background is gray

    {background-color: #333333;}

    Default font

    Text is dark black, 14 pixels

    p { font-size: 14px; color: black;}

    Header

    The header section will be changing positioned underneath the navigation of each page with the logo centered in the frame. The background is gray.

    Navigation

    The main navigation is 120px tall, positioned just above the header with a background color of Light Yellow and Dark Blue font color.

    nav { height: 120px; background-color: #ffffcc; color: #003366; }

    Images

    Images will be over light blue background with a darkblue border around the container

    figure { background-color: 6699CC; border: 2px solid #003366; }

    img { padding: 1em; }

    Links

    Links will be burgundy. On hover, they will have a light blue background and a blue border.

    a { color: #993333 }

    a:hover { background-color: #6699CC; border 2px dotted #336699; }
